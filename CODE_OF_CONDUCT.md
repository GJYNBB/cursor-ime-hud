# 行为准则

## 我们的承诺

我们希望 Cursor IME HUD 成为一个尊重他人、务实且欢迎贡献的项目。请以善意理解他人，耐心对待调试信息，并让讨论围绕项目本身展开。

## 鼓励的行为

以下行为有助于项目协作：

- 使用友善、包容的语言。
- 尊重不同的操作系统、工作流和经验背景。
- 报告问题时提供清晰的复现步骤和诊断信息。
- 以开放态度接受建设性反馈。
- 让功能讨论符合项目边界：只显示输入法状态，不读取用户内容，也不自动切换输入法。

## 不可接受的行为

以下行为不可接受，包括但不限于：

- 骚扰、侮辱、威胁或人身攻击
- 未经许可发布他人的私人信息
- 反复将 issue 或 pull request 引向项目范围之外
- 不遵循 [SECURITY.md](SECURITY.md) 的流程，在公开场合发布漏洞利用细节或敏感日志
- 其他由维护者合理认定为不适合专业开源项目的行为

## 执行方式

如果行为违反本准则，维护者可以编辑、隐藏或删除评论，关闭 issue 或 pull request，或限制相关参与者继续参与项目。

安全问题请使用 [SECURITY.md](SECURITY.md) 中的流程；普通项目支持请使用 [SUPPORT.md](SUPPORT.md) 和 GitHub issue 模板。

本政策的精神参考 [Contributor Covenant](https://www.contributor-covenant.org/)。
